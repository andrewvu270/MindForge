# MindForge - Learning Platform

## Overview
MindForge is a micro-learning platform designed for efficient knowledge acquisition across multiple fields. Perfect for busy professionals who want to learn quickly and readers who want to sharpen their minds through structured lessons and reinforcement quizzes.

## Learning Fields
- **Technology**: Latest in tech and AI developments
- **Finance**: Markets, investing, and economic principles  
- **Economics**: Economic trends and analysis
- **Culture**: Arts, society, and cultural developments
- **Influence Skills**: Communication and leadership
- **Global Events**: World news and political developments

## Tech Stack
- **Frontend**: React Native
- **Backend**: FastAPI (Python)
- **Database**: Supabase (PostgreSQL + Auth)
- **Payments**: Stripe
- **AI**: Hugging Face + OpenAI

## Core Features
- **Structured Lessons**: Bite-sized lessons with clear learning objectives
- **Reinforcement Quizzes**: Knowledge testing after each lesson with detailed feedback
- **Daily Challenges**: Curated learning paths for consistent progress
- **Progress Tracking**: Comprehensive analytics and achievement system
- **Field-Specific News**: Relevant news integrated with learning content
- **Multi-Field Learning**: Jump between different knowledge domains

## Learning Methodology
1. **Micro-Lessons**: 5-25 minute focused lessons
2. **Active Recall**: Reinforcement quizzes to solidify knowledge
3. **Spaced Repetition**: Daily challenges for long-term retention
4. **Cross-Domain Learning**: Ability to switch between fields for breadth
5. **Progress Analytics**: Detailed reports on learning performance
