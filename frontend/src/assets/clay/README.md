# Clay Mascot Images

This directory should contain the mascot images for each field:

- `tech-mascot.png` - Technology field mascot
- `finance-mascot.png` - Finance field mascot
- `econ-mascot.png` - Economics field mascot
- `culture-mascot.png` - Culture field mascot
- `influence-mascot.png` - Influence field mascot
- `global-mascot.png` - Global Events field mascot

## Generating Mascots

You can generate these mascots using:
1. The Hugging Face image generation service in the backend
2. Any AI image generator (DALL-E, Midjourney, Stable Diffusion)
3. Custom illustrations

## Fallback

If images are not available, the app will automatically use CSS-based clay character fallbacks.

## Image Requirements

- Format: PNG with transparent background
- Size: 512x512px recommended
- Style: Clay/3D character style matching the field theme
