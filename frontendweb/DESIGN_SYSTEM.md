# MindForge Design System

## Inspiration
Inspired by SuperHi, Godly, and Mobbin's award-winning designs.

## Core Principles
1. **Bold & Confident** - Large typography, strong hierarchy
2. **Playful Interactions** - Delightful micro-animations
3. **Asymmetric Balance** - Break the grid intentionally
4. **Color as Communication** - Strategic use of vibrant accents
5. **Generous Whitespace** - Let content breathe

## Typography Scale
- Hero: 72-96px (clamp)
- H1: 48-64px
- H2: 32-40px
- H3: 24-28px
- Body: 16-18px
- Small: 14px

## Color System
- Primary: #6366f1 (Indigo)
- Secondary: #8b5cf6 (Violet)
- Accent: #ec4899 (Pink)
- Success: #10b981 (Emerald)
- Warning: #f59e0b (Amber)
- Neutral: #64748b (Slate)

## Spacing Scale
- xs: 4px
- sm: 8px
- md: 16px
- lg: 24px
- xl: 32px
- 2xl: 48px
- 3xl: 64px
- 4xl: 96px

## Component Patterns
- Cards: Soft shadows, rounded corners (24px)
- Buttons: Bold, high contrast, clear hover states
- Forms: Large inputs, clear labels, inline validation
- Navigation: Sticky, minimal, context-aware

## Micro-interactions
- Hover: Scale (1.02-1.05), translate, shadow
- Click: Scale down (0.98), quick bounce back
- Load: Skeleton screens, progressive disclosure
- Success: Confetti, checkmarks, color transitions
